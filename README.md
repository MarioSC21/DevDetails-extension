# Pasos para usar la extension

1. Clonar el repositorio 
  ```https://github.com/MarioSC21/DevDetails-extension.git```
2. Ir al apartado de la gestion de extensiones del navegador, activar el modo desarrollador y darle click a **cargar descomprimida**
3. Dentro de **cargar descomprimida** seleccionar **devDownload**, **devExt**, **devScript**, se tienen que cargar las tres extensiones para el correcto funcionamiento.
4. Una vez cargado las tres carpetas levantamos la base de datos que esta dentro de la carpeta **devBaseDatos** para esto accedemos mediante la terminal a la carpeta y
realizamos los siguientes pasos:
    1. Instalamos las dependencias ```npm i```
    2. Levantamos el servidor con el comando ```npm run dev```
5. Una vez seguido los pasos anteriores accedemos a la pagina de DevDetalles nos logueamos, **Se tiene que tener una suscripcion**, y accedemos al video que se va a
descargar, la extension abrira pestanas y la cerrara para hacer la descarga
